# Force Sensor Using Magnetometer

## Project Overview

This project presents the design and development of a **magnetometer-based force sensor** capable of estimating applied force by measuring changes in the magnetic field. The system uses an **MLX90393 3-axis magnetometer** and permanent magnets, where the displacement caused by an external force results in a measurable variation in magnetic flux.

A custom mechanical housing was designed in **SolidWorks** to accurately position the sensor, magnets, and supporting components for reliable measurements. An **ESP32** microcontroller was used to acquire sensor data and enable force characterization through magnetic field analysis.

## Features

* Custom-designed sensor housing and assembly
* Force measurement using magnetic field variation
* MLX90393 3-axis magnetometer
* ESP32-based data acquisition
* CAD modeling and mechanical integration
* Experimental data collection and analysis

## Tools & Technologies

* SolidWorks
* ESP32
* MLX90393 Magnetometer
* Arduino IDE
* Embedded C/C++

---

*This project demonstrates the integration of mechanical design, embedded systems, and magnetic sensing to develop a compact, non-contact force measurement system.*
